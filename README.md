# Tailwind Personal Website

Personal website for Mohammad Dori using TailwindCSS.

<br>

# How to Use

Open `index.html` with your browser!

# Generate Styles

```bash
npx tailwindcss -i ./input.css -o ./assets/styles.css --watch
```
